# Pipeline Agent Report ŌĆö 2026-04-03

## Research Summary

- **USDA FDC API**: Rate limit is 1000 req/hr/IP. Best practice: use `requireAllWords` first, fall back to relaxed search. Our implementation already follows this pattern.
- **BullMQ Flows**: Parent waits for all children by default. `failParentOnFailure: true` opt-in. `getChildrenValues()` retrieves child results. Our FlowProducer usage is correct.
- **Vision Nutrition (DietAI24)**: Injecting USDA calorie values INTO prompt context before asking for estimates reduces MAE by 63-83%. Current implementation does post-hoc USDA cross-referencing ŌĆö works well enough, DietAI24 RAG is a future optimization.
- **Ensemble Confidence**: Original formula `base * (1 - 1/sqrt(n))` for n=2 yields 0.29x which is disastrously low. Research suggests logarithmic confidence scaling.

## Changes Applied

### 1. Menu Crawler ŌĆö Missing `glp1_labeled` in placeholder dietary flags (HIGH)
**File**: `src/lib/agents/menu-crawler/index.ts:127-131`
**Problem**: When LLM parse fails, placeholder entries omit `glp1_labeled` field, causing downstream type errors.
**Fix**: Added `glp1_labeled: false` to the fallback dietary flags object.

### 2. Menu Crawler ŌĆö Unsafe `menuSource` type assertion (HIGH)
**File**: `src/lib/agents/menu-crawler/index.ts:285`
**Problem**: `usedSource as "website" | "google_photos" | "manual"` but `usedSource` can be `"none"` when no menu source worked but we still reach the DB update (because early return at line 220 only fires when rawItems is empty).
**Fix**: Added guard: `usedSource === "none" ? undefined : usedSource as ...`

### 3. USDA Client ŌĆö Unsafe non-null assertion on `pickBestMatch()` (HIGH)
**File**: `src/lib/usda/client.ts:445`
**Problem**: `pickBestMatch(allResults)!` uses non-null assertion, but `pickBestMatch` can return `null` when allResults is empty (shouldn't happen given the guard above, but defense in depth).
**Fix**: Added explicit null check with descriptive error message.

### 4. Vision Analyzer ŌĆö Invalid array mutation in outlier detection (HIGH)
**File**: `src/lib/agents/vision-analyzer/index.ts:299-304`
**Problem**: `validAnalyses` is `const` array with `.push()` + `outlierIndices.length = 0` which is an unusual and fragile array truncation pattern.
**Fix**: Changed to `let` with reassignment and a separate `effectiveOutliers` variable.

### 5. Vision Analyzer ŌĆö Ensemble confidence formula too aggressive (MEDIUM)
**File**: `src/lib/agents/vision-analyzer/index.ts:322`
**Problem**: `base * (1 - 1/sqrt(n))` for n=2 gives 0.29x, making 2-photo analysis nearly always get skipped by the 0.4 confidence threshold in photo-worker.
**Fix**: Changed to `base * (1 + log2(n)/10)` capped at 1.0 ŌĆö gives ~7% boost for 2 photos, ~10% for 3, scales reasonably.

### 6. Menu Crawler Sources ŌĆö JSON-LD price double-formatting (MEDIUM)
**File**: `src/lib/agents/menu-crawler/sources.ts:123-131, 149-156`
**Problem**: Price formatting `$${price}` would produce `$$12.99` if the JSON-LD already includes a `$` prefix.
**Fix**: Added `startsWith("$")` check before prepending dollar sign in both Restaurant and Menu JSON-LD parsing.

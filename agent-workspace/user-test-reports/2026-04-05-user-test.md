## Cross-Persona Patterns

### Systemic Issues (affect all personas)
1. **Default browse returns only beverages** — the `isDishWorthRecommending()` filter is not applied at query time; 9 drink records dominate unfiltered searches
2. **`macro_embedding` column missing** — breaks `/similar`, some sort paths, vector search
3. **Dish name contamination** — price + description concatenated into name field during crawl
4. **Photo files deleted** — v2 photos removed from disk but still referenced in DB

### Improvements Since Last Run (2026-04-04)
- Pad Thai correctly blocked for nut_free (was passing at 0.95 on 2026-04-02)
- `protein_min` and `calorie_limit` filters now enforce correctly (were leaking on 2026-04-02)
- Warnings correctly passed through to DishCard (was stripped in page.tsx mapping)
- Distance sort and radius filter working accurately
